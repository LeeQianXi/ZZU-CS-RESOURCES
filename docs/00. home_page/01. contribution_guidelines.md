## 1. 本地构建

1.克隆本项目仓库

```python
git clone https://github.com/yzbaaa/ZZU-CS-RESOURCES.git
cd ZZU-CS-RESOURCES
```

2.安装 python 依赖

```python
pip install -r requirements.txt
```

3.启动 mkdocs 本地服务

```python
mkdocs serve
```

之后可以通过访问 localhost:8000 预览网站。
